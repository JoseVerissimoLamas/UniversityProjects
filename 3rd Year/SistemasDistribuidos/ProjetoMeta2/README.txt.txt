Para correr o projeto é necessário iniciar os seguintes programas pela seguinte ordem:

Primeiro: 
Correr o seguinte programa: RMI SERVER(RMIServer.java) que está na pasta \ucBusca\src\ucBusca\meta1;

Segundo:
Correr o Servidor Multicast (MulticastServerB.java) que está na pasta \ucBusca\src\ucBusca\meta1;

Terceiro (necessário ter o tomcat instalado):
Correr o web browser. 

Quarto:
Correr o rmi Client se assim desejar.

Após estes passos o browser irá abrir no seu motor de busca default.
