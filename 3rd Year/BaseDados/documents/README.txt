---------
 READ ME 
---------

1º - Seguir os passos de instalação da framework Django 2.2.7
2º - Seguir os passos de instalação da dependência de crispy_forms para Django (template de bootstrap4)
3º - Com o terminal de python aberto na diretoria do projeto, escrever o comando "python manage.py runserver". Isto irá iniciar o servidor localhost no porto 8000, onde estará o website pronto a ser usado.

Nota: Como a base de dados foi colocada online, não terá de se preocupar muito. Apenas tem que se certificar que está na rede do DEI (ou usar um VPN para a rede do DEI), pois a base de dados encontra-se na máquina presente na DEICloud da Ana Raquel.
